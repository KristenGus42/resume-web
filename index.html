Billboard

The project my team did for the 2022-2023 school year was titled Billboard. This project was my first formal introduction to professional collaboration and leadership. My team and I used this as an opportunity to learn web development skills and API integration.

Billboard is a web based trivia app that used the user's spotify data to create questions based on their specific music taste. Some skills we learned through this implementation include using APIs, using O-Auth as well as integrating code with hosting services, in particular we attempted to use firebase. Unfortunatley because of Spotify's developer's restrictions we are not allowed to officially publish our website. Below is some images and videos showcasing the website, alongside the github.
